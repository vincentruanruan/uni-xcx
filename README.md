# uni-xcx

## 全局安装vue-cli
```
npm install -g @vue/cli
```

## 创建uni-app
```
vue create -p dcloudio/uni-preset-vue my-project
```

## 运行并发布uni-app
```
npm run dev:%PLATFORM%
npm run build:%PLATFORM%
```

## %PLATFORM% 可取值如下：
```
值          平台
h5          H5
mp-alipay   支付宝小程序
mp-baidu    百度小程序
mp-weixin   微信小程序
```
